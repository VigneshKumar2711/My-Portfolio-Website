# 🎯 Personal Portfolio Website

This is a simple and responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. It showcases my projects, skills, resume, and provides a way to contact me.

## 📌 Description

The portfolio includes:

- **Introduction** – Short personal intro
- **Projects** – A list of sample works or links to real projects
- **Skills** – Technologies I’m familiar with
- **Resume** – Downloadable resume file
- **Contact** – Email or contact info for collaboration

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)

frontend developer portfolio
